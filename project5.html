Advanced Painting Software

Introduction

TThis project presents an advanced 2D interactive image editor designed to offer a robust set of features for image manipulation and enhancement. The editor enables users to split and combine multiple images, compute histogram equalization for both grayscale and RGB channels, and apply a variety of filters to improve image quality. Additional capabilities include bit-plane slicing, multiple edge detection techniques, and image contouring processes. Users can manually fine-tune image filters to suit their preferences and compare edited images with the original, with an option to toggle this comparison on or off. To further assist users in understanding the effects of different filters, the editor includes a comparison function that highlights distinctions between filter types, ensuring a clear and intuitive editing experience.

Strength & Uniqueness

In order to build a user-friendly interface, I added more menu for user can use the function in my editor based on each category. For example, when user just want to do drawing and painting, user can choose paint from menu home and the functions of paint will be displayed. But when user want to do filtering image, a different function menu will be replaced the previous menu. This ensure that user will not get confuse with all the functions in my editor.

Next, an overall comparison among all the filtered and edited image is available in this editor so that user does not need to keep on changing the filtering to see the differences. User can turn on and off any time during editing image to compare the differences. Besides auto filtering, user can also filter image manually by adjusting the value with scale bar provided.
