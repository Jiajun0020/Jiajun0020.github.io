DICOM: Medical Data Visualizer

Introduction

Visualization is key in scientific computing, turning complex data into clear visuals. The Visualization Toolkit (VTK), an open-source platform, excels in creating interactive 3D graphics. This assignment uses VTK to build a custom DICOM medical data visualizer, enhancing medical imaging analysis for better diagnosis and treatment. The tool leverages VTK’s advanced rendering and interaction features to help healthcare professionals explore medical data effectively. This report examines the visualizer’s design, implementation, and impact, highlighting VTK’s potential to transform medical data into insightful visualizations.

Flow Diagram

System Architecture

The DICOM medical data visualizer, built with VTK for rendering and PyQt5 for its interface, enables users to load, render, and interact with DICOM images and 3D volumes using VTK readers. Users can import DICOM files or folders via a button in the interface, after which patient details—name, ID, birthday, and gender—are extracted and displayed. The visualizer supports viewing individual .dcm files from a dataset with vtkImageViewer2 and allows length measurements by drawing lines on images using mouse events.

It also generates a 3D volume from the DICOM dataset using vtkImageData output and enhances edges with gradient magnitude from the reader’s GetOutputPort. Bone and skin extraction from the 3D volume is achieved with vtkFlyingEdges3D, while multiplanar reconstruction lets users view the series in sagittal, coronal, and axial directions. Additional features include preset window width/level adjustments and the ability to save modified image viewers, making this tool versatile for medical data analysis.

Strength & Uniqueness

This DICOM Medical Data Visualizer has the following strength and uniqueness.

  1. It can import a single .dcm file or a folder for DICOM data sets.
  2. It can display DICOM files in an image viewer and show them slice by slice.
  3. Able to perform edge enhancements on DICOM images.
  4. Able to display a 3D volume of DICOM data.
  5. Show bone of the 3D volume.
  6. Show skin of the 3D volume.
  7. Able to show DICOM images in multiplanar reconstruction views.
  8. Able to draw lines and measure length.
  9. Read DICOM files to retrieve patient information.
  10. Able to save DICOM images and export them in PNG file format.
